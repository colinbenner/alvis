Move elements If this icon is active you can move one or more nodes. In order to do so, just select one of the nodes with your mouse or draw a frame around several nodes and relocate the group by moving one of the nodes.
A right click brings you back to this mode from every other mode.		 right click
Add a node Add a node to the graph by clicking left. n
Add a connection In this mode you can connect two nodes with an edge by left clicking on one after the other.
When shift is pressed during clicking, second node becomes new first node for next connection		 c
Remove a node or a connection Removes a single connection or a node with all its connections. del (when sth.
is highlighted)
Change the layout Tries to arrange the graph as good as possible.  
Fit to Page Scales the graph to fit the size of the graph editor. Essential, if you zoomed in to much an parts of the graph are gone.
