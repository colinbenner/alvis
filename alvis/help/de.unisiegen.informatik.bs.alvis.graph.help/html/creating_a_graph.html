Hot Key
Bewegt Elemente Wenn dieses Symbol aktiv ist, kann man einzelne oder mehrere Knoten des Graphen verschieben. Dazu klickt man entweder einen Knoten direkt an oder zieht einen Rahmen um mehrere und klickt danach einen der gelb markierten Knoten an und zieht ihn/sie an die gewünschte Stelle.
Mit einem Rechtsklick kann man aus jedem anderen Modus in diesen zurückkehren.		  
Füge Knoten hinzu Fügt Knoten mit aufsteigender Nummer per Linksklick an den gewünschten Stellen zum Graphen hinzu. n
Füge Kante hinzu Ist dieser Modus aktiv, kann man zwei Knoten mit einer Kante verbinden, indem man sie nacheinander anklickt. c
Löscht einen Knoten oder eine Kante Löscht eine einzelne Kante oder einen Knoten und alle seine Kanten.  
Änder das Layout Versucht den Graph möglichst übersichtlich im Fenster anzuordnen.  
fitToPage ;-) Skaliert den Graph auf die Fenstergröße. Sehr praktisch, wenn ein Graph durch heranzoomen teilweise aus dem Sichtfeld verschwunden ist.  
Vergrößern Zoomt an der Position des Cursors herein. Strg + Mausrad hoch
Verkleinern Zoomt an der Position des Cursors heraus Strg + Mausrad runter
Füge einen Graphen hinzu Erzeugt einen zufälligen Graphen. Gewählt wird zwischen einem Zyklus und einem Baum, der eine festgelegte Tiefe und eine durchschnittliche Anzahl an Kindknoten pro Knoten hat.  
Lösche alle Knoten im Graph Löscht den gesamten Graphen, auch alles, was im Moment nicht sichtbar ist.
