The Algorithm-Editor

Algorithms using the Alvis language can be written with the Algorithm-Editor. The editor supports the user by highlighting source code's syntax, by offering established completions and by highlighting incorrect code. Common shortcuts, like "Ctrl + c" or "Ctrl + v" for copy and paste text can also be used in this editor.

  • The Autocompletion
  • Highlighting mistakes inside source code
  • Changing font size and style

The Autocompletion

In the editor code can be completed by using the short key "Ctrl + space bar". Already defined variables, common keywords and types known by the compiler will be autocompleted. Furthermore, the types' methods and attributes can be completed. A "()" is appended to methods, so one can identify them as functions, which are being autocompleted.

Highlighting mistakes inside source code

Mistakes inside the source code are being highlighted while doing a 0.5 long break from typing. When going to the end of the row which contains the error or by rolling the mouse pointer over the fault position, one can get support.

Changing font size and style

The font size and style can be changed by using the buttons at the toolbar. One can only see them, while an algorithm editor is active.


Explanation of the buttons:

Symbol

Meaning

Line numbers

Line numbers can be shown or hidden by right-clicking on the left border and choosing "....".
