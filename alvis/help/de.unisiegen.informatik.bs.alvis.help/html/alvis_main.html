Alvis

Welcome to Alvis, the tool for visualization of algorithms developed at the University of Siegen.
Alvis is an environment meant for developing algorithms in pseudo code and seeing what they do by using them on example data. By experimenting with different examples and algorithms you deepen your knowledge. Es soll eine Umgebung erstellt werden, in der man Algorithmen in einer Pseudosprache angeben und diese Algorithmen auf Beispieldaten anwenden kann. Dabei sollen die Beispieldaten möglichst leicht zu erstellen sein, z.B. sollte ein Graph für einen Graphenalgorithmus, einfach zusammengeklickt werden können. Das macht dem Nutzer das Ausprobieren von selbst ausgedachteen Algorithem sehr einfach, da nur die AlVis Anwendung benötigt wird, um einen Algorithmus auf die Beispieldaten anzuwenden. Die Idee hinter dem ganzen ist es, sich die Algorithmen dann Stück für Stück anzeigen zu lassen und synchron dazu die Änderungen an den Beispieldaten mit zu verfolgen. So sieht man graphisch, wie der geschriebene Algorithmus auf den Daten arbeitet und kann so eventuell eingeschlichene Denkfehler leicht finden und beheben. Die erstellten Algorithmen und Beispieldaten sollten abgespeichert und exportiert werden können. Die Anwendung wird komplett modular aufgebaut, sodass die Anwendung nicht nur Graphenalgorithmen spezialisiert wird, sondern später durch Plugins für andere Algorithmenarten erweitert werden kann. Ein Beispiel für eine andere Algorithmenart, könnten die Synchronisationsprobleme aus den Betriebssystemen sein. So ist das Ziel, die Anwendung durch den modularen Aufbau so "allgemein" zu halten, dass sie für alle möglichen Algorithmen erweitert werden kann.
