The Run

The so called run is the core of Alvis. By default the left view shows the algorithm and the right view shows your example. You control the run with the buttons in the above bar.

Button

Tool-Tip

Description
Start run (Re)start the run with the given parameters und delete all existing decisions.
Previous Break Point Takes the algorithm back to the previous break point.
Next Break Point Lets the algorithm advance to the next break point. Break points are the end of the lines with a ";" at the end.

If the algorithm is at a break point the current line is highlighted in orange. At a decision point the highlight is blue, a decision points demands a decision from the user.
