Der Ablauf

Der Ablauf, auch "Run" genannt, ist das Herzstück von Alvis. Standardmäßig sieht man im linken View den Algorithmus und im rechten das Beispiel. Mit den Buttons steuert man den Ablauf.

Button

Tool-Tip

Bedeutung
Start run Starte den Ablauf mit den gegebenen Parametern von vorne und lösche alle schon getroffenen Entscheidungen
Vorheriger Haltepunkt Lässt den Algorithmus zum vorherigen Haltepunkt zurücklaufen.
Nächster Haltepunkt Lässt den Algorithmus bis zum nächsten Haltepunkt laufen. Haltepunkte sind das Ende der mit ";" markierten Zeilen

Befindet sich der Algorithmus an einem Haltepunkt, wird die aktuelle Zeile orange hinterlegt. Bei einem Entscheidungspunkt (Decision Point) ist die Hinterlegung blau, hier wird eine Entscheidung vom Benutzer benötigt. file:///home/basti/Programmierung/alvis1074/de.unisiegen.informatik.bs.alvis.help/nl/de/html/run.html
