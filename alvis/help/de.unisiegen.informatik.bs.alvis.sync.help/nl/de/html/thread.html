Prozess

Ein Prozess entspricht einem laufenden Programm. Da ein Computer mehrere Prozesse "gleichzeitig" ausführen soll, pausiert die CPU nach einiger Zeit den ausgeführten Prozess und ruft einen anderen auf. So entsteht eine Quasi-parallele Ausführung auf Systemen, dessen CPU-Anzahl weit unter der der Anzahl Prozesse liegt.
Da ein Prozess keinen Einfluss darauf hat, wann die CPU ihm Rechenzeit zuteilt oder entzieht, er aber verschiedene Ressourcen im Zugriff haben kann, muss der Zugriff auf diese Ressourcen synchronisiert werden.
Zur Visualisierung von Prozessen stellt diess Plugin den Datentyp Thread zur Verfügung, von denen jeder einen vom Benutzer definierten Algorithmus ausführt.
