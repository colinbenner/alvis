Puffer

Ein Puffer ist ein Datentyp, mit dem mehrere Prozesse untereinander kommunizieren und arbeiten können. Der Puffer stellt einen sog. "kritischen Abschnitt" dar, der nie von mehr als einem Prozess gleichzeitig verwendet werden darf um Dateninkosistenzen zu vermeiden. Diese Bedingung kann durch verschiedene Mechanismen synchronisiert werden und stellt ein beliebtes Beispiel der Prozesssynchronisation dar. Gleichzeitig hat ein Puffer in der Realität nur begrenzten Platz, so dass auch Schreibzugriffe auf einen vollen Puffer und Lesezugriffe auf einen leeren Puffer verhindert werden müssen.
Damit diese Beispiele mit Alvis visualisierbar sind, enthält dieses Plugin eine Nachbildung eines Puffers (Buffer) mit wählbarer Kapazität. Zum Einfügen und Löschen eines Inhalts stehen die Methoden insert() und remove() zur Verfügung.
