Algorithmus definieren

Wenn der Benutzer ein Beispiel mit dem Scenario Editor erstellt hat fehlt noch der Algorithmus/die Algorithmen. Für jeden Prozess wird ein Algorithmus definiert, den er im Run ausführen wird.
Da der Editor und dessen Bedienung schon in Abschnitt Alvis -> Algorithmen schreiben beschrieben wurde wird an dieser Stelle darauf verzichtet.

Beispiel

Das folgende Beispiel zeigt die Algorithmen für das aus dem Abschnitt Semaphor bekannte Beispiel, in dem zwei Prozesse streng abwechselnd einen String ausgeben. 
main(Scenario s) begin
	Thread List tl = s.threads;
	for Thread t in tl: begin
		t.start(t.algo),
	end
end

Prozess1(Semaphore s1, Semaphore s2) begin
	while true: begin
		s1.P();
		print("Prozess 1");
		s2.V();
	end
end

Prozess2(Semaphore s1, Semaphore s2) begin
	while true: begin
		s2.P();
		print("Prozess 2");
		s1.V();
	end
end
